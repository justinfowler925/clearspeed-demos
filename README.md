# Affinity Insurance Demo Site

Demo website for Affinity Insurance Company featuring auto, home, life, and workers compensation insurance products with ClearSpeed voice verification integration.

## 🚀 Live Site

**www.clearspeeddemos.com**

## 📦 Tech Stack

- React 18 + Vite
- Tailwind CSS v4
- Radix UI + shadcn/ui
- GitHub Pages

## 🎯 Features

- **Auto Insurance**: Quote flows and theft claim forms
- **Home Insurance**: Quote flows and storm/water damage claims  
- **Life Insurance**: Quote flows with health questionnaires
- **Workers Comp**: Premium audit flows
- **Voice Verification**: ClearSpeed integration

## 🔧 Deploy

```bash
pnpm install
pnpm run build
pnpm run deploy
```
